# Chromosome-level NIPT reference frame

A typed tabular training frame derived from a `NIPTControlGroup`. Each
row corresponds to one reference sample and includes chromosome-level
read counts and fractions for chromosomes `1:22`, `X`, and `Y`.

## Usage

``` r
NIPTReferenceFrame(.data)
```

## Arguments

- .data:

  Data frame payload for the typed reference frame constructor.

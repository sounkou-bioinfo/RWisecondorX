# WisecondorX prediction result

A list-like S7 object produced by
[`rwisecondorx_predict()`](https://sounkou-bioinfo.github.io/RWisecondorX/reference/rwisecondorx_predict.md).

## Usage

``` r
WisecondorXPrediction(.data)
```

## Arguments

- .data:

  Named list payload for the typed constructor.

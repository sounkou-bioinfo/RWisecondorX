# WisecondorX reference panel

A list-like S7 object produced by
[`rwisecondorx_newref()`](https://sounkou-bioinfo.github.io/RWisecondorX/reference/rwisecondorx_newref.md).
It contains the autosomal branch plus optional female (`.F`) and male
(`.M`) gonosomal branches.

## Usage

``` r
WisecondorXReference(.data)
```

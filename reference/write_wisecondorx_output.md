# Write prediction output files

Writes bins BED, segments BED, aberrations BED, and statistics text
file.

## Usage

``` r
write_wisecondorx_output(prediction, outprefix)
```

## Arguments

- prediction:

  A `WisecondorXPrediction` object.

- outprefix:

  Path prefix for output files.

## Value

`outprefix` (invisibly).
